# A-love (知心) 项目

一个温暖可爱的心理健康前端应用，通过数字化手段为用户提供情感支持和日常治愈体验。

## 技术栈

- HTML5, CSS3 (Grid, Flexbox, Variables)
- JavaScript (ES6+)
- Vue.js 2.6.14 (CDN)

## 功能模块

### 核心功能
- **首页** - 导航栏、主题切换、功能入口卡片、底部导航
- **每日一句** - 便签风格展示随机治愈系句子
- **趣味小测试** - 测试功能框架（开发中）
- **夸夸卡** - 信封式交互，随机展示鼓励句子
- **情绪工具箱** - 六种情绪调节方法卡片
- **心情记录** - 心情日历、颜色选择、记录管理

### 特色功能
- 双主题支持（浅色/深色）
- 点击爱心/星星粒子特效
- 顶部随机暖心句子
- 底部全局导航栏
- 夸夸卡状态记忆功能

## 运行方式

```bash
# Python
python -m http.server 8080

# Node.js
npx http-server -p 8080
```

或直接使用浏览器打开 `index.html`

## 设计理念

- **温暖**：采用粉色、蓝色等柔和色调
- **可爱**：圆润的卡片设计，SVG图标
- **治愈**：精选的文案内容，传递正能量
- **易用**：简洁的交互，清晰的功能分区
