# A-love (知心)

心理健康前端应用，提供情感支持和日常治愈体验。

## 技术栈

- HTML5, CSS3 (Grid, Flexbox, Variables)
- JavaScript (ES6+)
- Vue.js 2.6.14 (CDN)

## 功能模块

### 首页
导航栏、主题切换、功能入口卡片、本月心情轨迹日历、底部导航

### 每日一句
便签风格展示随机治愈系句子

### 趣味小测试
- **水果人格测试** - 12道选择题，探索性格特质（草莓/橙子/葡萄型人格）
- **心理压力自测** - 8道选择题，评估压力等级（低/中/较高/高压力）
- **每日塔罗占卜** - 随机抽牌获取今日指引（8张塔罗牌）

### 夸夸卡
信封式交互，随机展示鼓励句子，支持状态记忆

### 情绪工具箱
六种情绪调节方法卡片（深呼吸、冥想、运动、艺术、音乐、倾诉）

### 心情记录
- **7种情绪颜色**：开心(粉)、平静(蓝)、兴奋(黄)、焦虑(紫)、放松(绿)、难过(矢车菊蓝)、生气(红)
- 心情描述输入框
- 心情记录列表（支持月份切换查看）
- GitHub风格的心情颜色日历
- 数据导出/导入功能（JSON格式备份恢复）
- localStorage本地存储

## 运行方式

```bash
# Python
python -m http.server 8080

# Node.js
npx http-server -p 8080

# 或直接打开 index.html
```

## 特色功能

- **双主题支持** - 浅色/深色模式，localStorage持久化
- **点击粒子特效** - 爱心/星星动画效果
- **顶部随机句子** - 暖心文案展示
- **底部全局导航** - 所有页面统一的导航入口
- **状态记忆** - 夸咔卡开启状态、主题偏好等
- **数据管理** - 心情记录支持导出/导入JSON备份
- **月份导航** - 心情记录列表支持按月筛选查看
