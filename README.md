# 知心 - 温暖你的每一天

一个温暖可爱的心理健康前端应用，提供每日一句、趣味测试、夸夸卡和情绪工具箱等功能。纯使用ai进行编写qwq

## 项目特点

- **双主题支持**：浅色和深色两种模式，深色模式采用温暖配色，不死板
- **响应式设计**：完美支持电脑端和移动端
- **点击特效**：点击页面时产生爱心、星星等粒子效果
- **分层架构**：清晰的代码结构，易于维护和扩展
- **可爱风格**：圆滑卡片设计，温暖可爱的视觉体验

## 技术栈

- HTML5
- CSS3（CSS变量、Grid布局、Flexbox）
- JavaScript（原生JS）
- Vue.js 2.6.14（CDN引入）

## 使用说明

### 本地运行

1. 直接双击 `index.html` 在浏览器中打开
2. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8080
   
   # 使用 Node.js
   npx http-server -p 8080
   ```
3. 访问 `http://localhost:8080`

### 主题切换

点击右上角的月亮/太阳图标即可在浅色和深色模式之间切换。主题偏好会自动保存。

## 扩展指南

### 添加新的每日一句

编辑 `src/js/everyday.js` 文件，在 `quotes` 数组中添加新句子：

```javascript
const quotes = [
    "现有的句子...",
    "你的新句子..."  // 添加这里
];
```

### 添加新的夸夸句子

编辑 `src/js/praise.js` 文件，在 `praises` 数组中添加新句子：

```javascript
const praises = [
    "现有的句子...",
    "你的新句子..."  // 添加这里
];
```

### 修改情绪工具箱内容

编辑 `src/js/toolbox.js` 文件，在 `toolContents` 对象中修改对应工具的内容。

### 添加新功能页面

1. 在 `src/pages/` 下创建新文件夹
2. 创建对应的 HTML、CSS、JS 文件
3. 在首页添加新的卡片入口
4. 在 `src/js/main.js` 的 `navigateTo` 函数中添加路由
